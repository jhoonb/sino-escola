### Instalando

Windows:

Python 3.6 >
Python pip
lib: schedule, pygame


Linux:

```
sudo apt-get update
sudo apt-get install python3
sudo apt-get install python3-pip
```

Após instalar o Python e o Pip, instale as libs:

```
python3 -m pip install pygame
python -m pip install schedule
```
